# Phase 2 可视化平台

> **Visualization Platform for Stream Processing**

## 概述

本目录包含20个交互式可视化组件，帮助理解和监控流处理系统。

## 组件分类

### 知识可视化 (4个)

- 知识图谱浏览器
- 证明树可视化
- 决策向导
- 学习路径推荐

### 监控可视化 (6个)

- 实时监控Dashboard
- Metrics Explorer
- Log Viewer
- Alert Manager
- Checkpoint Analyzer
- State Inspector

### 开发与调试 (4个)

- 代码沙盒
- API Playground
- Query Builder
- Config Editor

### 分析工具 (6个)

- Topology Viewer
- Performance Analyzer
- Job Comparison
- Data Explorer

## 技术栈

- HTML5/CSS3/JavaScript
- Mermaid.js 图表
- 响应式设计
- 深色/浅色主题

## 使用方法

直接在浏览器中打开对应的 HTML 文件即可使用。
