DataStream.map() - 数据映射转换

对DataStream中的每个元素应用映射函数，返回新的DataStream。

DataStream<Integer> numbers = env.fromElements(1, 2, 3, 4, 5);
DataStream<Integer> squared = numbers.map(x -> x * x);
squared.print();
