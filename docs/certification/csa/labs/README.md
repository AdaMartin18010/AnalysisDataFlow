## 实验环境准备

### 必需软件

- Java 8 或 11
- Maven 3.6+
- Docker & Docker Compose
- IntelliJ IDEA / VS Code

### 快速启动

```bash
# 1. 克隆实验环境仓库
git clone https://github.com/analysisdataflow/labs-environment.git
cd labs-environment

# 2. 启动基础服务(Kafka, MySQL, Redis)
docker-compose up -d

# 3. 验证环境
./scripts/verify-env.sh
```
