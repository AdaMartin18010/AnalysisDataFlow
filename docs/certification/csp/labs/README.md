申请免费实验环境：

- 地址: <https://labs.analysisdataflow.org/csp>
- 有效期: 7 天
- 配置: 3 节点 K8s 集群 + Flink Operator

```bash

# - MySQL: localhost:3306
# - Redis: localhost:6379
# - Prometheus: localhost:9090
# - Grafana: localhost:3000
```
