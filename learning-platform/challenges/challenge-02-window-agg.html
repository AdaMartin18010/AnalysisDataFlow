挑战2: 窗口聚合优化

优化复杂的窗口聚合操作，处理迟到数据和增量计算

📝 挑战描述

中级

在实际生产环境中，窗口聚合面临数据倾斜、迟到数据和性能问题。本挑战要求你实现一个优化的窗口聚合方案，解决这些常见问题。

业务场景

实时统计每个商品类目的订单金额，窗口大小为1分钟，需要：

  • 使用两阶段聚合解决数据倾斜问题
  • 实现增量聚合优化性能
  • 处理迟到数据 (允许5分钟迟到)
  • 迟到数据输出到侧输出流
  • 支持 Session Window 动态合并

🏗️ 两阶段聚合原理

┌─────────────────────────────────────────────────────────────────┐
│                     两阶段聚合 (解决数据倾斜)                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   原始数据流                                                      │
│      │                                                          │
│      ▼                                                          │
│   ┌──────────┐    第一阶段: 随机预聚合                             │
│   │ Map      │    key = random(0-99) + "_" + category             │
│   │ (预聚合)  │    → 局部聚合，分散热点                            │
│   └────┬─────┘                                                  │
│        │                                                        │
│   ┌────▼─────┐    窗口聚合                                        │
│   │ Window   │    按随机key聚合                                   │
│   │ Aggregate│                                                   │
│   └────┬─────┘                                                  │
│        │                                                        │
│   ┌────▼─────┐    第二阶段: 去除随机前缀                            │
│   │ Map      │    key = original category                         │
│   │ (去前缀)  │    → 提取原始key                                   │
│   └────┬─────┘                                                  │
│        │                                                        │
│   ┌────▼─────┐    最终聚合                                        │
│   │ Final    │    按原始key最终汇总                                │
│   │ Aggregate│                                                   │
│   └──────────┘                                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

💡 提示

提示 1: 两阶段聚合实现

第一阶段使用 key.hashCode() % 100 + "_" + key 生成随机前缀。

这样可以将有相同 key 的数据分散到 100 个不同的子任务中。

提示 2: AggregateFunction

实现四个方法：

  • createAccumulator(): 创建累加器
  • add(value, accumulator): 添加元素到累加器
  • getResult(accumulator): 从累加器提取结果
  • merge(a, b): 合并两个累加器 (用于会话窗口)

提示 3: 迟到数据处理

使用 .allowedLateness(Time.minutes(5)) 允许迟到。

使用 .sideOutputLateData(LATE_DATA_TAG) 将迟到数据发送到侧输出流。

完整实现代码

import org.apache.flink.api.common.eventtime.WatermarkStrategy;
import org.apache.flink.api.common.functions.AggregateFunction;
import org.apache.flink.api.java.tuple.Tuple2;
import org.apache.flink.streaming.api.datastream.DataStream;
import org.apache.flink.streaming.api.datastream.SingleOutputStreamOperator;
import org.apache.flink.streaming.api.environment.StreamExecutionEnvironment;
import org.apache.flink.streaming.api.windowing.assigners.TumblingEventTimeWindows;
import org.apache.flink.streaming.api.windowing.time.Time;
import org.apache.flink.util.OutputTag;

import java.time.Duration;

public class OptimizedWindowAggregation {
    private static final OutputTag<OrderEvent> LATE_DATA_TAG = 
        new OutputTag<OrderEvent>("late-data"){};
    
    public static void main(String[] args) throws Exception {
        StreamExecutionEnvironment env = 
            StreamExecutionEnvironment.getExecutionEnvironment();
        
        // 1. 创建带 Watermark 的数据流
        DataStream<OrderEvent> orders = env.fromElements(
                new OrderEvent("A001", "electronics", 100.0, 1000L),
                new OrderEvent("A002", "electronics", 200.0, 2000L),
                new OrderEvent("A003", "clothing", 50.0, 3000L)
            )
            .assignTimestampsAndWatermarks(
                WatermarkStrategy.<OrderEvent>forBoundedOutOfOrderness(
                    Duration.ofSeconds(5))
                .withTimestampAssigner((event, timestamp) -> event.timestamp)
            );
        
        // 2. 两阶段聚合
        SingleOutputStreamOperator<CategoryStats> result = orders
            // 第一阶段: 随机预聚合
            .map(event -> {
                int randomPrefix = event.category.hashCode() % 100;
                String prefixedKey = randomPrefix + "_" + event.category;
                return Tuple2.of(prefixedKey, event);
            })
            .keyBy(t -> t.f0)
            .window(TumblingEventTimeWindows.of(Time.minutes(1)))
            .aggregate(new SumAggregate())
            // 第二阶段: 去除前缀后最终聚合
            .map(stats -> {
                String category = stats.category.substring(stats.category.indexOf("_") + 1);
                return new CategoryStats(category, stats.totalAmount, stats.count);
            })
            .keyBy(s -> s.category)
            .window(TumblingEventTimeWindows.of(Time.minutes(1)))
            .allowedLateness(Time.minutes(5))
            .sideOutputLateData(LATE_DATA_TAG)
            .aggregate(new FinalSumAggregate());
        
        // 3. 获取迟到数据
        DataStream<OrderEvent> lateData = result.getSideOutput(LATE_DATA_TAG);
        
        result.print("main");
        lateData.print("late");
        
        env.execute();
    }
    
    // 第一阶段聚合函数
    public static class SumAggregate implements 
        AggregateFunction<Tuple2<String, OrderEvent>, Acc, CategoryStats> {
        
        @Override
        public Acc createAccumulator() {
            return new Acc(null, 0.0, 0);
        }
        
        @Override
        public Acc add(Tuple2<String, OrderEvent> value, Acc accumulator) {
            if (accumulator.category == null) {
                accumulator.category = value.f0;
            }
            accumulator.totalAmount += value.f1.amount;
            accumulator.count++;
            return accumulator;
        }
        
        @Override
        public CategoryStats getResult(Acc accumulator) {
            return new CategoryStats(accumulator.category, 
                                     accumulator.totalAmount, accumulator.count);
        }
        
        @Override
        public Acc merge(Acc a, Acc b) {
            a.totalAmount += b.totalAmount;
            a.count += b.count;
            return a;
        }
    }
    
    // 第二阶段聚合函数
    public static class FinalSumAggregate implements 
        AggregateFunction<CategoryStats, Acc, CategoryStats> {
        
        @Override
        public Acc createAccumulator() {
            return new Acc(null, 0.0, 0);
        }
        
        @Override
        public Acc add(CategoryStats value, Acc accumulator) {
            if (accumulator.category == null) {
                accumulator.category = value.category;
            }
            accumulator.totalAmount += value.totalAmount;
            accumulator.count += value.count;
            return accumulator;
        }
        
        @Override
        public CategoryStats getResult(Acc accumulator) {
            return new CategoryStats(accumulator.category, 
                                     accumulator.totalAmount, accumulator.count);
        }
        
        @Override
        public Acc merge(Acc a, Acc b) {
            a.totalAmount += b.totalAmount;
            a.count += b.count;
            return a;
        }
    }
    
    public static class Acc {
        String category;
        double totalAmount;
        long count;
        
        Acc(String category, double totalAmount, long count) {
            this.category = category;
            this.totalAmount = totalAmount;
            this.count = count;
        }
    }
    
    public static class OrderEvent {
        String orderId;
        String category;
        double amount;
        long timestamp;
        
        OrderEvent() {}
        
        OrderEvent(String orderId, String category, double amount, long timestamp) {
            this.orderId = orderId;
            this.category = category;
            this.amount = amount;
            this.timestamp = timestamp;
        }
        
        @Override
        public String toString() {
            return String.format("OrderEvent{orderId='%s', category='%s', amount=%.2f}",
                orderId, category, amount);
        }
    }
    
    public static class CategoryStats {
        String category;
        double totalAmount;
        long count;
        
        CategoryStats(String category, double totalAmount, long count) {
            this.category = category;
            this.totalAmount = totalAmount;
            this.count = count;
        }
        
        @Override
        public String toString() {
            return String.format("CategoryStats{category='%s', totalAmount=%.2f, count=%d}",
                category, totalAmount, count);
        }
    }
}
