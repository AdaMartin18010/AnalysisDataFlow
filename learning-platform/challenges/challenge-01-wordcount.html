挑战1: WordCount 实现

流处理领域的 "Hello World" - 实现实时词频统计

📝 挑战描述

初级

WordCount 是流处理领域最经典的入门程序。你的任务是实现一个 Flink 作业，从数据源读取文本，实时统计每个单词出现的次数。

功能要求

  • 从 Socket (端口 9999) 或 Kafka Topic 读取文本数据
  • 将文本分割为单词，统一转换为小写
  • 每5秒输出一次词频统计结果
  • 支持处理乱序数据 (使用 Watermark)
  • 结果输出到控制台和文件

💡 提示

提示 1: 创建数据源

使用 env.socketTextStream("localhost", 9999) 创建 Socket 数据源。

测试前需要先启动 nc -lk 9999。

提示 2: 分词逻辑

使用正则表达式 \\W+ 分割非单词字符。

过滤空字符串，统一转换为小写。

提示 3: 窗口聚合

使用 keyBy(0) 按单词分组。

使用 sum(1) 对计数求和。

完整实现代码

import org.apache.flink.api.common.eventtime.WatermarkStrategy;
import org.apache.flink.api.common.functions.FlatMapFunction;
import org.apache.flink.api.java.tuple.Tuple2;
import org.apache.flink.streaming.api.datastream.DataStream;
import org.apache.flink.streaming.api.environment.StreamExecutionEnvironment;
import org.apache.flink.streaming.api.windowing.assigners.TumblingProcessingTimeWindows;
import org.apache.flink.streaming.api.windowing.time.Time;
import org.apache.flink.util.Collector;

public class WordCount {
    public static void main(String[] args) throws Exception {
        // 1. 创建执行环境
        StreamExecutionEnvironment env = 
            StreamExecutionEnvironment.getExecutionEnvironment();
        env.setParallelism(1);
        
        // 2. 创建数据源 (Socket)
        DataStream<String> text = env.socketTextStream("localhost", 9999);
        
        // 3. 分词并转换
        DataStream<Tuple2<String, Integer>> wordCounts = text
            .flatMap(new Tokenizer())
            .returns(new TypeHint<Tuple2<String, Integer>>() {});
        
        // 4. 分组聚合
        DataStream<Tuple2<String, Integer>> result = wordCounts
            .keyBy(value -> value.f0)
            .window(TumblingProcessingTimeWindows.of(Time.seconds(5)))
            .sum(1);
        
        // 5. 输出结果
        result.print();
        
        // 6. 执行作业
        env.execute("Streaming WordCount");
    }
    
    public static class Tokenizer implements FlatMapFunction<String, Tuple2<String, Integer>> {
        @Override
        public void flatMap(String value, Collector<Tuple2<String, Integer>> out) {
            // 统一转小写，按非单词字符分割
            String[] words = value.toLowerCase().split("\\W+");
            
            for (String word : words) {
                if (word.length() > 0) {
                    out.collect(new Tuple2<>(word, 1));
                }
            }
        }
    }
}

进阶版本 (带 Watermark)

// 支持 Event Time 的版本
DataStream<Tuple2<String, Integer>> result = text
    .assignTimestampsAndWatermarks(
        WatermarkStrategy.<String>forMonotonousTimestamps()
            .withTimestampAssigner((event, timestamp) -> System.currentTimeMillis())
    )
    .flatMap(new Tokenizer())
    .keyBy(value -> value.f0)
    .window(TumblingEventTimeWindows.of(Time.seconds(5)))
    .sum(1);
