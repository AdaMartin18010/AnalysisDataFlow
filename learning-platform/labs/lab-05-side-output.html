实验5: 侧输出流与多路复用

通过可视化演示理解侧输出流(Side Output)机制，学习多路数据分流与处理

📖 实验原理

什么是侧输出流 (Side Output)?

侧输出流允许将数据流拆分为多个输出流，常用于：

  • 处理迟到数据 (Late Data)
  • 错误数据分流 (Error Handling)
  • 数据分类路由 (Data Routing)
  • 多维度统计 (Multi-dimension Analytics)

// 定义侧输出流标签
OutputTag<Event> lateDataTag = new OutputTag<Event>("late-data"){};
OutputTag<Event> errorDataTag = new OutputTag<Event>("error-data"){};

SingleOutputStreamOperator<Result> mainStream = stream
    .process(new ProcessFunction<Event, Result>() {
        @Override
        public void processElement(Event event, Context ctx, 
                                   Collector<Result> out) {
            // 验证数据
            if (!isValid(event)) {
                ctx.output(errorDataTag, event);
                return;
            }
            
            // 检查是否迟到
            if (event.getTimestamp() < ctx.timerService().currentWatermark()) {
                ctx.output(lateDataTag, event);
            } else {
                out.collect(processEvent(event));
            }
        }
    });

// 获取侧输出流
DataStream<Event> lateStream = mainStream.getSideOutput(lateDataTag);
DataStream<Event> errorStream = mainStream.getSideOutput(errorDataTag);
