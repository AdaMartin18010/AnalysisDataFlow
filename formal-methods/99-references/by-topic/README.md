## 使用指南

### 选择合适主题

- **模型检测**: 关注自动验证、状态空间探索、反例生成
- **定理证明**: 关注严格数学证明、无限状态系统、证明助手
- **进程代数**: 关注并发理论、进程组合、行为等价
- **分布式系统**: 关注一致性、容错、分布式算法

### 交叉引用

主题之间存在紧密联系：

```mermaid
graph TB
    PA[进程代数] --> DS[分布式系统]
    PA --> TP[定理证明]
    MC[模型检测] --> DS
    MC --> PA
    TP --> DS
    TP --> PA
```
