## 🔧 邮件发送工具功能

### 核心功能

```
email-sender.py
├── 解析功能
│   ├── 解析Markdown邮件文件
│   ├── 提取主题、正文、附件
│   └── 识别收件人信息
├── 个性化功能
│   ├── 变量替换
│   ├── 称呼定制
│   └── 内容定制
├── 发送功能
│   ├── 单封发送
│   ├── 批量发送（带间隔）
│   ├── 附件处理
│   └── Markdown转HTML
├── 记录功能
│   ├── 发送状态记录
│   ├── 错误日志
│   └── 统计报告
└── 错误处理
    ├── 自动重试（最多3次）
    ├── 重试间隔
    └── 失败标记
```

### 使用方式

```bash
# 解析邮件模板
python advisory-board/email-sender.py parse emails/01-tyler-akidau-invitation.md

# 模拟发送（Dry Run）
python advisory-board/email-sender.py dry-run emails/01-tyler-akidau-invitation.md --email tyler@example.com

# 批量模拟发送
python advisory-board/email-sender.py dry-run-batch

# 实际发送单封
python advisory-board/email-sender.py send emails/01-tyler-akidau-invitation.md --email tyler@example.com

# 批量发送
python advisory-board/email-sender.py send-batch

# 查看状态
python advisory-board/email-sender.py status
```
