## 任务 2：Algolia DocSearch 申请

1. 打开浏览器访问 `https://docsearch.algolia.com/apply/`
2. 填写申请表单：
   - **Repository URL**: `https://github.com/<owner>/AnalysisDataFlow`
   - **Website URL**: `https://knowledge-graph.analysisdataflow.github.io/`
   - **Email**: 仓库维护者邮箱
   - **Project Name**: AnalysisDataFlow
   - **Is your project open source?**: Yes (Apache 2.0)
3. 提交申请，等待 Algolia 团队邮件回复（通常 1-3 个工作日）

### 收到凭证后操作

Algolia 会邮件发送 `appId`、`apiKey`、`indexName`。

更新以下文件中的配置占位符：

#### 文件 1：`KNOWLEDGE-GRAPH/index.html`

搜索 `ALGOLIA_CONFIG` 占位区，替换为：

```javascript
const ALGOLIA_CONFIG = {
  appId: 'YOUR_APP_ID',
  apiKey: 'YOUR_SEARCH_API_KEY',
  indexName: 'analysisdataflow',
  hitsPerPage: 10
};
```

> 注意：使用 **Search-Only API Key**，不要暴露 Admin API Key。

### 本地搜索降级

如果 Algolia 申请周期过长，当前 `index.html` 已内置本地搜索降级逻辑，无需额外操作。
