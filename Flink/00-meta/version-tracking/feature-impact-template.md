# 特性影响分析模板

> 所属阶段: Flink/version-tracking | 用途: 评估新特性对现有文档的影响

---

## 使用方法

1. **复制此模板** 到新文件：`Flink/version-tracking/feature-impact-{flip-number}.md`
2. **填写所有字段**（标记为 `[必填]` 的字段）
3. **评估影响范围** 并更新检查清单
4. **提交到版本控制** 并关联相关任务
