## 🔧 修复建议

```bash
python .scripts/fix-external-links.py --mode fix-redirects
```

2. **404错误**: 检查链接是否拼写正确

### Archive.org备份

- 直接访问: `https://web.archive.org/web/*/{URL}`
- 使用工具批量查询: `python .scripts/fix-external-links.py --mode find-archive`
