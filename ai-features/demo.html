💻 命令行使用

所有AI功能都可以通过命令行脚本使用：

# 文档摘要 python scripts/ai-features/document-summarizer.py --input README.md python scripts/ai-features/document-summarizer.py --input Struct/ --batch --format json # 智能搜索 python scripts/ai-features/smart-search-indexer.py --build --source . python scripts/ai-features/smart-search-indexer.py --search "checkpoint mechanism" python scripts/ai-features/smart-search-indexer.py --semantic-search "流处理容错" # 问答机器人 python scripts/ai-features/qa-bot-knowledge-base.py --build --source . python scripts/ai-features/qa-bot-knowledge-base.py --ask "什么是Checkpoint?" # 学习路径 python scripts/ai-features/learning-path-personalizer.py \ --goal "成为Flink专家" \ --background "Java后端3年" \ --level intermediate
