# 项目统计数据

本目录包含 AnalysisDataFlow 项目的各类统计数据文件。

## 📁 文件说明

| 文件 | 说明 | 更新频率 |
|------|------|----------|
| `feedback-stats.json` | 社区反馈统计数据 | 每月（自动化） |
| `contributor-stats.json` | 贡献者统计数据 | 每月（自动化） |

## 🔒 数据用途

这些数据文件用于：

1. 生成反馈报告和仪表板
2. 追踪社区活跃度
3. 识别高优先级任务
4. 更新贡献者列表

## 📝 数据格式

详见 [reports/feedback/README.md](../reports/feedback/README.md)

## ⚠️ 注意事项

- 本目录中的文件由自动化脚本生成和更新
- 请勿手动修改这些文件
- 如需自定义统计，请修改相应的脚本
