# 社区行为准则 (Code of Conduct)

> **版本**: v1.0 | **生效日期**: 2026-04-08 | **适用范围**: AnalysisDataFlow 社区所有参与者

---

## 我们的承诺

作为 AnalysisDataFlow 社区的成员、贡献者和维护者，我们致力于为我们的社区提供一个友好、包容、尊重的环境。我们承诺确保无论年龄、体型、残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何，每个人的参与体验都是无骚扰的。

我们承诺以有助于建立开放、友善、多元、包容和健康社区的方式行事和互动。

---

## 我们的标准

有助于为我们的社区创造积极环境的行为包括：

### ✅ 鼓励的行为

| 行为 | 说明 |
|-----|------|
| **尊重他人** | 尊重不同的观点、经验和背景 |
| **友善包容** | 使用友善、包容的语言，欢迎新成员 |
| **建设性反馈** | 提出建设性的批评，并接受他人的建设性反馈 |
| **关注社区** | 关注什么对社区最有益 |
| **表现同理心** | 对其他社区成员表示同理心 |
| **知识分享** | 乐于分享知识和经验，帮助他人成长 |
| **耐心指导** | 对新手保持耐心，提供指导和支持 |
| **承认错误** | 勇于承认错误，从中学习 |

### ❌ 不可接受的行为

| 行为 | 说明 |
|-----|------|
| **歧视性言行** | 使用歧视性语言或进行歧视性行为 |
| **侮辱攻击** | 发表侮辱性、贬损性评论或进行人身攻击 |
| **骚扰行为** | 任何形式的骚扰，包括公开或私下的不当行为 |
| **隐私侵犯** | 未经明确许可发布他人私人信息 |
| **不当行为** | 在专业环境中可被合理认为不适当的行为 |
| **恶意破坏** | 故意破坏、spam 或恶意行为 |
| **trolling** | 故意引发争议或冲突的行为 |

## 责任与执行

### 社区维护者职责

社区维护者有责任明确行为准则的标准，并预期会公正、一致地对任何不可接受的行为采取适当和公平的纠正措施。

维护者有权和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、问题和其他贡献，并在适当情况下暂时或永久禁止任何贡献者从事他们认为不适当、威胁、冒犯或有害的行为。

### 执行流程

#### 第一步：提醒与教育

对于初犯或轻微违规，维护者将：

- 私下联系相关人员
- 解释其行为如何违反准则
- 提供改进建议
- 给予改正机会

#### 第二步：警告

对于重复违规或较严重违规：

- 发出正式警告
- 记录违规行为
- 要求立即停止不当行为

#### 第三步：临时封禁

对于持续违规或严重违规：

- 临时限制参与社区活动（如 7-30 天）
- 禁止参与特定功能（如评论、提交 PR）

#### 第四步：永久封禁

对于极端违规或屡教不改：

- 永久禁止参与社区
- 移除所有贡献权限

### 报告指南

如果您遭遇或目睹了不可接受的行为，或有任何其他顾虑，请通过以下方式报告：

| 报告渠道 | 联系方式 | 适用情况 |
|---------|---------|---------|
| **GitHub** | 私信项目维护者 | 一般性问题 |
| **邮件** | <conduct@analysisdataflow.org> | 需要隐私保护的情况 |
| **匿名报告** | 通过 [匿名表单] | 希望保持匿名 |

所有报告将被保密处理。我们将在 3 个工作日内回复，并在必要时进行调查。

## 适用范围

本行为准则适用于所有社区空间，也适用于当个人在公共空间代表本社区时。代表本社区的示例包括使用官方电子邮件地址、通过官方社交媒体账户发布内容，或在线上或线下活动中担任指定代表。

### 适用场景

- ✅ GitHub Issues 和 Pull Requests
- ✅ GitHub Discussions
- ✅ 社区会议和活动
- ✅ 官方社交媒体和通讯
- ✅ 一对一的社区相关沟通
- ✅ 代表社区参与的外部活动

## 冲突解决

### 争议处理原则

1. **善意假设**：首先假设各方都是出于善意
2. **直接沟通**：鼓励当事人之间首先尝试直接沟通解决
3. **第三方介入**：如无法解决，可请求维护者介入
4. **社区讨论**：必要时在社区公开讨论（尊重隐私的前提下）
5. **最终决定**：维护者做出最终决定，社区成员应尊重决定

### 冲突解决步骤

```

│   发现问题      │

    ┌────────┐

    └────┬───┘      ▼
      是 │    ┌─────────────────┐

         │    └────────┬────────┘
         │             ▼
         │    ┌─────────────────┐
         │    │   调查与调解    │
         │    └────────┬────────┘
         │             ▼

         │    └─────────────────┘
         │

    │ 结束   │
    └────────┘
```
