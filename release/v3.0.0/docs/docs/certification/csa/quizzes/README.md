## 使用建议

### 学习阶段

1. **模块学习期**: 完成每个模块后，做对应章节练习题
2. **复习巩固期**: 针对错题进行专项练习
3. **考前冲刺期**: 完成3套模拟考试

### 目标正确率

| 阶段 | 目标正确率 | 说明 |
|------|------------|------|
| 模块练习 | 70%+ | 理解为主，不必强求 |
| 复习巩固 | 85%+ | 查漏补缺 |
| 模拟考试 | 80%+ | 达到考试水平 |
