# Phase 2 术语表

> **Phase 2 Glossary**

## 形式化证明术语

| 英文 | 中文 | 说明 |
|------|------|------|
| Formal Proof | 形式化证明 | 使用数学方法严格验证系统正确性 |
| TLA+ | TLA+ | 时序逻辑规范语言 |
| Coq | Coq | 交互式定理证明助手 |
| Invariant | 不变量 | 系统执行过程中保持为真的性质 |
| Liveness | 活性 | 系统最终会发生的性质 |
| Safety | 安全性 | 系统不会发生的错误性质 |

## 流处理术语

| 英文 | 中文 | 说明 |
|------|------|------|
| Stream Processing | 流处理 | 实时处理连续数据流的技术 |
| Watermark | 水印 | 事件时间的进度标记 |
| Checkpoint | 检查点 | 系统状态的周期性快照 |
| Exactly-Once | 精确一次 | 每条记录只处理一次的语义保证 |
| Window | 窗口 | 对数据流进行分组的时间区间 |

## 一致性术语

| 英文 | 中文 | 说明 |
|------|------|------|
| Consistency | 一致性 | 数据在不同副本间的同步程度 |
| Causal Consistency | 因果一致性 | 因果关系的事件顺序保证 |
| Linearizability | 线性一致性 | 强一致性模型 |
| Eventual Consistency | 最终一致性 | 弱一致性模型 |
