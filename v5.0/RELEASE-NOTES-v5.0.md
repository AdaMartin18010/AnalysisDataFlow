## ⚠️ 兼容性说明

### 浏览器支持

| 浏览器 | 最低版本 | 状态 |
|--------|----------|------|
| Chrome | 90+ | ✅ 完全支持 |
| Firefox | 88+ | ✅ 完全支持 |
| Safari | 14+ | ✅ 完全支持 |
| Edge | 90+ | ✅ 完全支持 |

### 已知限制

1. **IE 11**: 不再支持
2. **旧版移动端浏览器**: 知识图谱功能受限
3. **离线访问**: 需要Service Worker支持

### 迁移指南

从v4.0升级到v5.0无需任何操作。所有已有链接自动重定向到新地址。
