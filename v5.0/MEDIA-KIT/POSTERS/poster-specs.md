## 设计规范

### 字体

| 用途 | 字体 | 大小 |
|------|------|------|
| 标题 | Inter Bold | 72-120px |
| 副标题 | Inter SemiBold | 36-48px |
| 正文 | Inter Regular | 24-32px |
| 数据 | Inter Bold | 96-144px |

### 颜色

| 用途 | 色值 |
|------|------|
| 背景 | #1A1A2E |
| 主色 | #0066CC |
| 强调 | #00A3E0 |
| 文字 | #FFFFFF |
| 辅助文字 | #B0B0C0 |

### Logo使用

- 完整Logo，高度不低于100px
- 周围保留安全空间
- 使用反色版本（白色）
