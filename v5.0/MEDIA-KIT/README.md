## 🎨 品牌要素

### 主色调

| 颜色 | 色值 | 用途 |
|------|------|------|
| **主色** | `#0066CC` | Logo、主按钮、链接 |
| **辅助色** | `#00A3E0` | 高亮、强调 |
| **成功色** | `#28A745` | 成功状态、正面信息 |
| **警告色** | `#FFC107` | 警告、注意 |
| **深色** | `#1A1A2E` | 深色模式背景 |
| **浅色** | `#FFFFFF` | 浅色背景 |

### 字体

- **标题**: Inter, system-ui, sans-serif
- **正文**: Inter, system-ui, sans-serif
- **代码**: JetBrains Mono, Consolas, monospace

### Logo使用规范

✅ **正确用法**:

- 保持Logo比例，不拉伸
- 使用官方提供的颜色
- 周围保留足够的留白空间

❌ **错误用法**:

- 修改Logo颜色
- 拉伸或压缩Logo
- 添加阴影或效果
- 在低对比度背景上使用
