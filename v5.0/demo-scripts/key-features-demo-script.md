讲解: "复杂的示例可以在浏览器中直接运行"

示例代码:
```java
StreamExecutionEnvironment env =
    StreamExecutionEnvironment.getExecutionEnvironment();

env.enableCheckpointing(60000);
env.getCheckpointConfig().setCheckpointingMode(
    CheckpointingMode.EXACTLY_ONCE);

DataStream<Event> stream = env
    .fromSource(source, WatermarkStrategy.forBoundedOutOfOrderness(
        Duration.ofSeconds(5)), "source")
    .keyBy(Event::getUserId)
    .window(TumblingEventTimeWindows.of(Time.minutes(1)))
    .aggregate(new CountAggregate());

stream.print();
env.execute("Checkpoint Demo");
```

展示:

- 代码在云端执行
- 实时输出结果
- 执行时间统计
- 错误提示（如有）

```

### Q&A 常见问题准备

**Q1: 学习平台是否免费？**
> A: 是的，所有课程和实验都是免费的。未来可能会推出认证考试的付费服务。

**Q2: 英文翻译质量如何保证？**
> A: 我们有专业的翻译团队和术语审核流程，确保翻译质量。

**Q3: 可以离线使用吗？**
> A: 可以克隆GitHub仓库到本地使用，但在线功能需要联网。

**Q4: 如何贡献内容？**
> A: 查看 CONTRIBUTING.md，我们欢迎各种形式的贡献。

**Q5: 移动端体验如何？**
> A: 所有功能都针对移动端进行了优化，可以随时随地学习。
