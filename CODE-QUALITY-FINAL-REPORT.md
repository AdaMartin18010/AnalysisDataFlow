## 最佳实践总结

   ```python
   def process(data: Dict[str, Any]) -> List[Result]:
       ...
   ```

   ```python
   # ✅ 推荐
   user_profiles = load_user_data()

   # ❌ 避免
   ```

   ```python
   def calculate():

       计算指标
   ```
