# 社区治理模型

## 治理结构

### 核心团队

- **项目维护者**: 负责整体方向
- **技术委员会**: 负责技术决策
- **社区经理**: 负责社区运营

### 贡献者角色

1. **Visitor**: 访问者
2. **Contributor**: 贡献者
3. **Committer**: 提交者
4. **Maintainer**: 维护者

## 决策流程

### 技术决策

1. 提案 (Proposal)
2. 讨论 (Discussion)
3. 投票 (Voting)
4. 实施 (Implementation)

### 社区决策

1. 收集反馈
2. 公开讨论
3. 核心团队决策
4. 执行并通知

## 行为准则

### 我们的承诺

- 开放和包容
- 尊重和礼貌
- 协作和共享

### 不可接受的行为

- 骚扰和歧视
- 侮辱和攻击
- 破坏性行为
